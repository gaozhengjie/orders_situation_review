# 订单复盘分析
## 功能描述
用于代驾订单复盘，辅助老胡的日常分析工作

## 实现方法
- 前后端不分离（因为分离的我不会）
- 后端采用的fastapi
- jinja2渲染前端页面
- 数据库mongodb
